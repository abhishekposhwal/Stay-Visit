# **App Name**: StayVis

## Core Features:

- Property Listing Import: Allow users to import property listing data from supported platforms like Airbnb or Zillow, capturing details like location, price, amenities, and photos.
- AI-Powered Summary Generation: Utilize an AI tool to generate a concise summary of property listings, highlighting key features, nearby attractions, and unique selling points. This will allow potential travelers to quickly understand the listing at a glance.
- Interactive Map View: Integrate an interactive map that displays property listings based on user search criteria (location, price range, amenities). Enable users to filter listings on the map dynamically.
- Wishlist Creation and Management: Enable users to create wishlists to save their favorite property listings for later viewing and consideration.
- Personalized Recommendation: Make personalized listing recommendations, by comparing the AI summaries for un-starred properties, with those of the user's stared, saved listings.
- User Account Management: Allow users to create and manage their profiles, wishlists, and view their saved searches.
- Intuitive Listing Card UI: Design visually appealing listing cards that display essential property information such as photos, price, location, and a short AI-generated summary. Implement clear typography and intuitive layout to enhance readability.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) to evoke a sense of relaxation and tranquility, complementing the travel focus.
- Background color: Light gray (#F0F0F0), providing a clean and neutral canvas.
- Accent color: Muted Teal (#80CBC4) to highlight key interactive elements like buttons and links.
- Headline font: 'Playfair', serif, to give an elegant, fashionable, high-end feel.
- Body font: 'PT Sans', sans-serif, used with 'Playfair' to ensure the app body is readable.
- Use clear, minimalist icons that reflect property details such as bedrooms, bathrooms, and amenities. Ensure icons are consistent in style and weight for visual harmony.
- Implement a grid-based layout with ample whitespace to ensure content is easily scannable. Prioritize visual hierarchy to guide users through the interface effectively.
- Incorporate subtle transitions and animations (e.g., hover effects on listing cards, smooth map interactions) to enhance user engagement without being distracting.
